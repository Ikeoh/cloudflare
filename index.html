Entendendo os Custos da Nuvem

Para uma decisão estratégica, é preciso compreender como cada componente impacta a fatura final.

Egresso (Saída de Dados)

É a cobrança sobre os dados que saem da nuvem para o usuário.

O Risco: "Taxa sobre o sucesso". Quanto mais usuários acessam nossos dados, mais caro fica. No R2, esta taxa é zero.

Vendor Lock-in

Dificuldade financeira de trocar de fornecedor no futuro devido às altas taxas de saída.

Prevenção: Escolher o R2 garante que nossos dados permaneçam portáteis e acessíveis sem "multas" de migração.

Escrita (Classe A)

Custo para inserir ou modificar dados.

Exemplos: Upload de ficheiros e criação de backups.

Leitura (Classe B)

Custo para visualizar ou consultar dados.

Exemplos: Abrir uma imagem ou gerar um relatório.
